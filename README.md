Resources, templates, code examples, and workflows supporting **FAIR**, reproducible, and high-quality marine research data management within the [Cluster of Excellence Ocean Floor](https://oceanfloor.marum.de/index.html).

Maintained by the **Data Stewardship Team**, this repository serves as a curated resource hub bringing together tools, templates, example code, and best practices to support researchers throughout the entire research lifecycle—from project planning to publication and long-term data preservation.

> 📖 **Looking for project documentation?**
>
> The primary documentation and knowledge base for Research Data Management in the Ocean Floor project is available in our **[Ocean Floor RDM Wiki](https://rdmwiki.uol.de/wiki)**.

---

# 🎯 Purpose

This repository aims to make research **more reproducible, reusable, and easier to start** by providing practical resources for researchers across the cluster.

It includes:

- 🛠️ Recommended research software
- 📊 Example analysis code and notebooks
- 📋 Metadata and documentation templates
- ♻️ FAIR data guidance
- 🔄 Research workflows
- 🎓 Tutorials and learning resources
- 📦 Reusable project templates
- 🔗 Curated links to community resources

This toolkit is actively maintained by the Data Stewardship Team and will continue to grow through contributions from the Ocean Floor research community.

---

# 👤 Who is this repository for?

This repository is intended for anyone working within or collaborating with the Ocean Floor Cluster, including:

- 🌊 Researchers
- 🎓 PhD candidates and early-career researchers
- 🗂️ Data stewards
- 🧪 Laboratory staff
- 🤝 Project partners and collaborators

Whether you are starting a new project, preparing data for publication, looking for coding examples, or searching for FAIR data management guidance, this repository is designed to support your research.

# 🚀 Getting Started

New to the Ocean Floor project? We recommend the following steps:

### 1. Learn about Research Data Management in Ocean Floor

Visit the **[Ocean Floor RDM Wiki](https://rdmwiki.uol.de/wiki)** for project-specific documentation, policies, and guidance.

### 2. Explore this repository

Browse the resources available in this repository, including:

- 📊 Code examples
- 🎓 Tutorials
- 📋 Templates
- 🛠️ Software recommendations
- 🔄 Research workflows
- 🔗 External resources

### 3. Use the ressources

Browse the files online or clone the repository to reuse code, templates, and workflows in your own research.

### 4. Contribute

Found a useful resource or noticed something that could be improved?

We welcome suggestions and contributions from the Ocean Floor community. You can submit improvements through GitHub Issues or Pull Requests. See our **[Contributing Guidelines](CONTRIBUTING.md)** to learn more.

### 5. Need help?

If you cannot find what you are looking for, consult the **Ocean Floor RDM Wiki** or contact the **Data Stewardship Team**.

# 🤝 Contributing to this Toolkit

This repository is a community resource and is actively maintained by the **Ocean Floor Data Stewardship Team**.
